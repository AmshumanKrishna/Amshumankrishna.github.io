E-commerce Microservices Platform

This project involved developing a scalable e-commerce platform using microservices architecture. The platform is designed to handle high traffic and integrates multiple features such as product catalog, user authentication, payment gateways, and order tracking. It employs Kafka for asynchronous communication and uses Docker for containerization.

Key Features:

  • Product Catalog Management
  • User Authentication and Authorization
  • Payment Gateway Integration
  • Order Tracking and Management
  • Scalable Architecture with Kafka
  • Containerization with Docker

Technologies Used:

  • Java
  • Spring Boot
  • Kafka
  • AWS
  • Docker
  • MySQL
